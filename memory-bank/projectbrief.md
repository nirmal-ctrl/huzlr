# Project Brief

## Overview
Standmate is an AI-powered standup and project management assistant. It is structured as a monorepo consisting of:
- A Next.js front-end (`standmate`)
- A FastAPI/Python back-end (`standmate-be`)

## Core Goals
- Streamline the agile standup process using AI
- Provide an intuitive and dynamic project management interface
- Seamlessly integrate with tools like Jira

## Scope
- Front-end user interface using modern React and Next.js capabilities.
- Back-end services for handling AI operations, database integrations, and overall application state.
- Integration endpoints to handle third-party platforms.
